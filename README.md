# ClapSwitch

A Python project that detects hand claps using a computer microphone and toggles a virtual ON/OFF switch.

## Features

- 🎙️ Real-time microphone monitoring
- 👏 Detects hand claps
- 💡 Toggles a virtual switch between ON and OFF
- ⚡ Built with Python, NumPy, and sounddevice

## Project Structure

```
ClapSwitch/
│── main.py
│── requirements.txt
│── README.md
│── .gitignore
└── venv/ (not uploaded to GitHub)
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ClapSwitch.git
```

2. Navigate to the project:

```bash
cd ClapSwitch
```

3. Install the dependencies:

```bash
pip install -r requirements.txt
```

## Running the Project

```bash
python main.py
```

## Expected Output

```
🎙️ Microphone is LIVE! Listening for claps...
💡 [SWITCH STATUS]: OFF
------------------------------------------
```
